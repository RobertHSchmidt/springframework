The flow definition registry subsystem for managing containers of flow definitions.

You can construct a generic, initially empty FlowRegistry, populate it with Flow definitions using a FlowRegistrar, then lookup flow definitions by id. For example: 

    // create registry
    FlowDefinitionRegistryImpl registry = new FlowDefinitionRegistryImpl();

    // populate registry
    ExternalizedFlowDefinitionRegistrar registrar = ..
    registrar.addLocation(...);
    registrar.addLocation(...);
    registrar.registerFlows(registry);

    // use registry
    FlowDefinition flow = registry.getFlow("myFlow");
