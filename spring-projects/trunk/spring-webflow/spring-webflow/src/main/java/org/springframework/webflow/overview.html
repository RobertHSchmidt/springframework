The root source code package of Spring Web Flow.

This package serves as the root and parent package for all Spring Web Flow sources. No source artifacts actually reside in this package.

Spring Web Flow's code base is structured into a set of logical layers. Higher layers depend on the layers directly beneath. Lower layers never depend on higher layers.
